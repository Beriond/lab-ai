# Pasta de Dados

Coloque aqui os arquivos **PDF** que serão indexados pelo módulo RAG.

O pipeline do módulo `2. RAG/` percorre esta pasta recursivamente e lê todos os arquivos `.pdf`.

## Sugestões de conteúdo para testar

- Artigos científicos
- Manuais ou políticas internas da empresa
- Capítulos de livros que você esteja estudando
- Material da própria aula

## Estrutura esperada

```
dados/
├── README.md           (este arquivo)
├── meu_artigo.pdf      (exemplo)
└── subpasta/
    └── outro_doc.pdf
```

> Não comite PDFs sensíveis ou com direitos autorais. Lembre-se: o curso é público.
